# 🛒 ASP.NET MVC E-Commerce Platform

A scalable, role-based e-commerce system built with **C# and ASP.NET MVC** using PostgreSQL as the database.
Designed following clean architecture principles with a focus on maintainability and scalability.

## 🚀 Project Overview

This project simulates a real-world e-commerce platform developed in **C#**,
including authentication, product management, shopping cart functionality,
and order processing modules.

---

## ✨ Features

- 🔐 User Registration & Authentication
- 👥 Role-Based Authorization (Admin / User)
- 📦 Product Management (CRUD Operations)
- 🛒 Shopping Cart System
- 📑 Order Management
- 📊 Admin Dashboard
- 📱 Responsive UI with Bootstrap

---

## 🏗 Architecture

The project follows a layered architecture:

- Controllers (Presentation Layer)
- Services (Business Logic Layer)
- Repositories (Data Access Layer)
- DTOs
- Entity Models
- Entity Framework Core Integration

This structure ensures separation of concerns and scalability.
